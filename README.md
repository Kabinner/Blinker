# Blinker
Lets you cast Blink by pressing Shift while using the movement keys.



### Optional Dependency
```
UnitXP_SP3 [https://github.com/allfoxwy/UnitXP_SP3]
Uses UnitXP_SP3 when available, as it consumes fewer resources.
```
